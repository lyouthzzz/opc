# 价值投资框架

本文件是价值投资 skill 的完整方法论参考，基于段永平/巴菲特的投资理念整理。本文件为零公司耦合的通用方法论，具体公司实例见 `companies/` 目录下各公司上下文文件。

---

## 一、投资哲学基础

### 核心三要素：Right Business, Right People, Right Price

巴菲特在午餐时告诉段永平的最重要的道理：投资公司最重要的是 right business, right people, right price。

段永平的升华：**right price 都可以不看**，因为通过商业模式和企业文化就足以过滤 99% 的公司。三要素的重要性排序为：right business ≈ right people > right price。

- **Right Business**：好生意。能持续赚钱、有护城河、能穿越周期的商业模式。
- **Right People**：好的人。管理层诚信、有能力、有长期主义思维、做对的事情。
- **Right Price**：好价格。价格合理，有安全边际。很重要但"不常见"。

### 本分与理性想长远

段永平投资体系的核心文化：

- **本分**：做该做的事情，不赚不该赚的钱。本分经营是对股东最好的回报。
- **理性想长远**：想长远所以必须理性，保持理性就必须有平常心，隔绝外部压力和诱惑。三者形成闭环。
- **做对的事情，把事情做对**：
  - "做对的事情"（do the right things）是方向性问题，最重要
  - "把事情做对"（do things right）是执行力问题
  - 做对的事情的重要性 > 今天融了多少钱

### 长期主义

- "相信时间的力量，慢慢来、慢慢做，但要做就要做好、做对"
- 投资买的是"未来的总量"——物理中的"速度 × 时间 = 总长度"
- 经济学的"速度"实际是物理的"加速度"——单位时间跑得更远，但投资看的是总量而非加速度
- 不要被短期增速变化迷惑：高增速是"加速度"，要看绝对值和长期趋势

### 消费者导向

三种企业导向的层级：
1. **利润导向**：错误，短视且危险
2. **客户导向（ToB）**：复杂
3. **消费者导向（ToC）**：最简单最纯粹——洞察最终用户需要什么，提供好的产品服务，商业闭环自然形成

## 二、看懂生意的方法论

### 什么是"看懂"

"看懂"不是知道公司做什么产品，而是理解：
1. 公司的本质是什么（不是什么）
2. 核心需求是什么（功能性 vs 情绪/精神）
3. 商业模式如何运转（钱从哪来到哪去）
4. 10 年后公司会怎样（可持续性）

### 看不懂可以 skip

- "人生很忙，看不懂的东西是可以 skip 的"
- 不必勉强投资看不懂的公司
- 但"看不懂"不等于"不去了解"——"看完所有能找到的资料"是基本要求

### 如何深入理解一家公司

1. **看完所有能找到的创始人/管理层采访**（包括视频）
2. **读完财报全文**，不只看摘要
3. **实地调研**：去门店看、去感受用户真实反应
4. **对比市场观点**：理解多空双方的逻辑
5. **用第一性原理拆解**：从本质出发，不被市场叙事带偏

### 破除常见误区

- **混淆品牌和品类**：消费者常把品牌当品类的代名词，这是品类创新者必须接受的荣耀和考验
- **用功能产品思维看情绪产品**：情绪价值产品的逻辑与功能产品不同
- **过度关注短期指引**：业绩指引对成长无益且有副作用，不给指引的公司往往更好
- **被"加速度"迷惑**：增速放缓 ≠ 业绩恶化，要看绝对值和趋势

## 三、五维度分析方法

### 维度一：商业模式（0-10 分）

**评估清单**：

| 指标 | 关注点 | 优秀标准 |
|------|--------|----------|
| 收入结构 | 产品/服务/授权占比，是否多元 | 多引擎驱动，不依赖单一产品 |
| 毛利率 | 水平及趋势 | >60% 且持续提升 |
| 净利率 | 水平及趋势 | >25% |
| 利润含金量 | 经营现金流 vs 净利润 | 现金流远超净利润 |
| 资产结构 | 有息负债率、现金储备 | 几乎零有息负债，现金充裕 |
| 资本回报 | ROE/ROIC | >30% |
| 定价权 | 价格敏感度、提价能力 | 用户价格不敏感 |
| 边际成本 | 成功后衍生品/授权的边际成本 | 边际成本递减 |

**评分标准**：
- 8-10 分：模式清晰、护城河可见、业绩稳健可预测、利润含金量极高
- 6-7 分：模式可行但有不确定性，业绩可预测性不够
- <6 分：模式脆弱、依赖偶然因素

**关键判断**：商业模式是否从"单一爆款/单一产品驱动"转向"系统平台驱动"？

### 维度二：市场规模（0-10 分）

**评估清单**：
- 行业总规模（TAM）及增速
- 公司市占率及 CR5 集中度
- 天花板距离：还有多少倍空间
- 国内 vs 海外收入占比及增速差异
- 增长驱动因素：扩张驱动 → 精细化运营驱动 → 平台驱动

**关键判断**：
- 国内市场是否从"扩张驱动"转向"精细化运营驱动"？
- 海外市场是"单产品出口"还是"平台全球化"？
- 市场增速放缓是自然规律还是竞争加剧？

### 维度三：竞争壁垒（0-10 分）

**护城河类型**：

1. **品牌独占性**："真正的强者具有独占性。品牌是种独占性"
2. **网络效应**：用户越多 → 数据越准 → 产品越好 → 用户更多
3. **规模优势**：供应链柔性、成本结构、渠道密度
4. **转换成本**：用户粘性、会员体系、收藏/沉没属性
5. **资源独占**：签约顶尖人才、核心位置门店、独占性资源

**黑洞效应**（最高级壁垒）：
- 头部资源主动向公司汇聚
- 新锐创作者/人才纷纷投奔
- 形成正反馈循环，后来者连入场资格都没有

**壁垒的局限性**：
- "壁垒不能保证需求永远存在，但可以让有需求的人持续关注公司"
- 壁垒 vs 持续性：壁垒已建起来，大家要顾虑的是持续性问题

### 维度四：企业文化（0-10 分）

**创始人画像维度**：
- 性格：沉稳、喜怒不形于色
- 能力：发现真实需求 + 敢于行动（最核心能力）
- 格局：不被沉没成本束缚、敢于否定自己
- 长期主义：相信时间力量、慢慢做但做好
- 清醒度：高光时主动降温、看到数字背后的陷阱

**文化信号**：
- 是否"做对的事情"优先于"做赚钱的事情"
- 是否消费者导向（而非利润导向）
- 是否用文化筛选人（招合适的人，而非只看能力）
- 高增长时是否主动"慢下来"（F1 维修站换胎）
- 是否尊重时间、尊重经营

**负面信号**：
- 为了热点而热点（如为了 AI 火就做 AI 产品，而非真正想清楚）
- 盲目大举投入新赛道
- 过度依赖业绩指引管理预期
- 被好消息冲昏头脑

### 维度五：估值（0-10 分）

**估值方法详解**：

#### 方法一：一眼胖瘦法

核心思想："用合理的价格买入一家伟大的公司，远胜于用便宜的价格买入一家平庸的公司"。

操作：
1. 不精算，先判断公司是"好公司"还是"平庸公司"
2. 若是好公司，判断当前价格是"明显便宜""合理"还是"明显贵"
3. 市场给低 PE 要辨别原因：是"真的贵"还是"市场太悲观"

#### 方法二："撇去爆款"模型

适用于有核心爆款/增长引擎的公司。

```
假设条件：
1. 剔除最核心爆款/增长引擎的全部利润
2. 企业不再成长（零增长）

计算：
- 爆款营收 × 净利率 = 爆款利润
- 总利润 - 爆款利润 = 剩余利润
- 当前市值 / 剩余利润 = 极端保守 PE

判断：
- 若极端保守 PE ≈ 合理 PE（如 25 倍）→ 市场过度悲观，存在安全边际
- 若极端保守 PE >> 合理 PE → 市场定价合理或偏贵
```

关键洞察：
- "市场给低 PE，不是因为它'贵'，而是因为它'太悲观'"
- 现实中爆款不会突然归零，其他业务仍在增长 → 安全边际更大
- 这个框架的价值在于"量化反推市场预期"
- 多引擎驱动的公司可对每个引擎分别做敏感性分析

#### 方法三：多情景估值

| 情景 | 假设 | 适用判断 |
|------|------|----------|
| 极端悲观 | 核心驱动归零 + 零增长 | 安全边际下限 |
| 悲观 | 核心驱动回落至爆发前 | 市场是否已反映 |
| 中性 | 管理层指引增速 | 合理定价 |
| 乐观 | 多引擎驱动 | 上行空间 |

对比当前市值在每种情景下的 PE，判断市场定价隐含了哪种预期。

#### 方法四：安全边际综合判断

- 当前 PE 隐含的市场预期是什么？
- 与公司历史估值对比（高位/低位）
- 与同业估值对比
- 机构一致预期及目标价
- "right price 很重要也不常见"——需要耐心等待

## 四、四种好运理论

来源于《佛畏系统》，用于评估企业核心竞争力的可持续性。

### 第一种：完全随机的好运

- **定义**：纯粹偶然、不可控且无法预测的运气
- **特征**：来得快去得快，一般不超过 6 个月
- **企业对应**：外部事件催化（如名人自发分享、政策红利、突发事件）
- **投资含义**：不可依赖，不能作为投资依据

### 第二种：数量保质量的好运

- **定义**：多产生作品/产品，用数量确保质量，尝试越多成功概率越大
- **企业实践**：
  - 小批量生产 → 测试市场反应 → 爆款追加资源、弱品直接砍掉
  - 储备大量创作者/产品线，形成"金字塔式梯队"
  - 从"赌一个"变成"测一批、选最优"
- **投资含义**：系统化的试错机制，成功率可提升

### 第三种：更高作品质量的好运

- **定义**：持续提高自身能力，用能力保质量
- **企业实践**：
  - 内部研发/设计团队辅助打磨产品
  - 分成/股权模式绑定长期合作
  - 提炼核心辨识度，工业化优化
- **投资含义**：能力建设是可持续的

### 第四种：专家身份让好运找上门

- **定义**：成为某个领域的专家/领导者后，机遇和资源主动来找你
- **企业实践**：
  - 行业共识：头部创作者/人才更愿意和领先公司合作
  - 正反馈循环：头部资源汇聚 → 产品更强 → 吸引更多资源
  - "黑洞效应"：行业最优质的资源持续向公司汇聚
- **投资含义**：最高级壁垒，一旦形成极难打破

### 综合判断标准

| 企业占据的好运类型 | 可持续性判断 |
|-------------------|-------------|
| 仅第一种 | 不可持续，成功不可复制 |
| 第一种 + 第二种 | 有一定持续性，但能力未固化 |
| 第二种 + 第三种 + 第四种 | 系统化可持续，核心竞争力已形成 |

"单个爆款有偶然性，但产品矩阵的持续产出能力已经形成"——这是第二+三+四种好运叠加的标志。

## 五、判断标准与决策框架

### 投资决策树

```
1. 是否看懂了这个生意？
   ├─ 否 → skip，不投资
   └─ 是 → 继续

2. Right Business？（商业模式 + 市场规模 + 竞争壁垒）
   ├─ 否 → 不投资
   └─ 是 → 继续

3. Right People？（企业文化 + 管理层）
   ├─ 否 → 不投资
   └─ 是 → 继续

4. Right Price？（估值 + 安全边际）
   ├─ 否 → 等待机会
   └─ 是 → 投资

5. 可持续性验证（四种好运）
   ├─ 仅依赖偶然 → 小仓位/观察
   └─ 系统化可持续 → 正常仓位
```

### 持续性争议的处理

- 持续性争议会长期存在——"在持续性上的争议会持续很久的"
- 企业需要"一直证明下去"——但"他们自己不需要证明什么，只要继续做该做的事情就好"
- "大家可以先看个五年十年吧"——给时间验证
- 用过往认知能力验证新领域——"我很好奇我对 X 的理解会不会在这里再次得到回报"

### 业绩指引的态度

- 永远不给预测，因为没有任何意义但早晚会有代价
- 巴菲特的 BRK 不给预测（甚至没有季度 conference call）
- 网易不给业绩指引，拼多多不给下季度/年度指引
- 指引的副作用：一旦达不到导致股价大跌 + 官司
- "每个季度或每年的业绩指引都是要花不少时间的，但对业绩成长却没有一丝一毫的作用"

### 关于做空与市场

- "这个世界上根本就没有什么恶意做空或者是恶意做多"
- "如果有哪家公司会因为被人做空而出现麻烦的话，那他本来就有麻烦"
- "每个公司的真正的买家就只有一个，就是公司自己，就是公司最后的现金流或未来的现金流"
- 5 年 10 年 15 年 20 年后，人们买公司是因为公司能赚到钱
