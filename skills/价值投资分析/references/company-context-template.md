# 公司知识库条目模板

本文件是用户私人公司知识库的条目模板。**仅当用户明确要求"把某公司沉淀进知识库"时**，才复制此文件为 `companies/<公司名>.md` 并填充内容。

日常价值分析不需要使用此模板——分析报告直接输出给用户即可，不写入知识库。
