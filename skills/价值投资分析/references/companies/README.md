# 公司知识库索引

本目录是**用户私人沉淀的公司理解知识库**，与分析流程相互独立。每个 `.md` 文件是用户对一家公司的深度理解，独立加载、互不耦合。

**重要**：这里存放的是用户长期积累的理解，不是每次分析的产物。日常分析直接用通用框架完成即可，不写入知识库。仅当用户明确要求"把某公司沉淀进知识库"时，才新增条目。

## 文件命名规范

- 使用公司英文名或拼音，小写，连字符分隔
- 示例：`popmart.md`、`xiaomi.md`、`maotai.md`、`wuxi-apptec.md`、`tencent.md`

## 使用说明

- 分析某公司时，先查本索引确认是否已收录
- **已收录** → 加载对应文件作为分析参照（创始人理解 + 投资人分析实例 + 评分参考）；**须对比元信息中的「数据截止」与 MCP 最新锚点**，滞后部分以 MCP 实时数据为准
- **未收录** → 直接用通用框架独立分析，无需做任何沉淀动作
- **多公司对比** → 可同时加载多个文件，但不将一家公司的结论套用到另一家

## 新增条目（仅用户主动发起）

仅当用户明确要求"把某公司沉淀进知识库"时：

1. 复制 `../company-context-template.md` 为本目录下的 `<公司名>.md`
2. 按模板结构填充内容
3. 更新本 README 的"已收录公司"表格
4. 若该公司在"待收录"列表中，将其移至"已收录"
