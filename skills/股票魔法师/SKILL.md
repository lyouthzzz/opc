# 股票魔法师

用这个 skill 把股票数据转成 SEPA / 股票魔法师风格的交易判断：是否可以买、是否继续观察、是否持有、是否减仓或卖出。

不要臆测缺失行情数据。用户没有提供当前价格、均线、成交量、基本面或消息面时，先获取可靠的最新数据；无法获取时，明确列出缺失项，并输出“数据不足”。

这是规则化交易框架，不是个性化投资建议。必须说明不确定性，不承诺收益。

数据统一来自同花顺问财：个股 / 指数行情、成交量、资金流、技术指标优先使用 `hithink-market-query`；行业强弱用 `hithink-industry-query`，板块强弱用 `hithink-sector-selector`。如果问财返回里缺少 SEPA 判断必需字段，例如均线、52 周高低点、相对强度或中枢点附近量价信息，不要补猜，直接输出“数据不足”。

## 参考手册

需要精确阈值、完整检查清单、A 股适配事项或交易日志模板时，读取 [references/sepa-strategy-manual.md](references/sepa-strategy-manual.md)。

普通买点、卖点、止损和仓位判断优先使用下面的核心规则。

## 必要输入

判断买点时，至少使用：

- 日线价格和成交量历史，足够计算 MA10 / MA20 / MA50 / MA150 / MA200，以及 52 周高低点。
- 相对强度排名，或市场/板块强弱排名。
- 形态信息：VCP 收缩次数、每次回撤幅度、中枢点价格、突破量比。
- 大盘和板块环境。
- 可用的基本面：营收增速、利润增速、EPS、机构资金、催化剂、重大风险。

判断持有或卖出时，还要使用：

- 买入价、仓位、初始止损、当前止损、买入后最高价、持有天数。
- 当前价相对 MA10 / MA20 / MA50、中枢点、近期波谷和 VCP 支撑的位置。
- 当前量价表现，尤其是放量滞涨、长上影、放量破位。

## 买入流程

1. 先排除不能买的股票：ST / *ST、退市风险、重大利空、A 股日均成交额低于 3 亿元、MA200 仍在下降、股价明显低于 MA50。
2. 检查趋势模板。要求股价高于 MA150 和 MA200，MA150 高于 MA200，MA200 高于 60 个交易日前，股价高于 MA50，股价较 52 周低点至少上涨 30%，位于 52 周高点 25% 以内，相对强度约在前 20%-30%。
3. 优先选择有基本面支撑的股票：净利润增速至少 25%，营收增速至少 20%，EPS 为正且增长，有机构增持、行业龙头地位或清晰催化剂。
4. 识别形态。优先 VCP：2-5 次收缩，每次回撤小于前一次，成交量逐步萎缩，最后一次收缩约 2%-5%，最后一次收缩上沿形成清晰中枢点。VCP 不明显时，才考虑杯柄、双底、高位旗形、3-C 或 shakeout。
5. 只有有效触发才买入：收盘价或明确价格行为突破中枢点，量比至少 1.5，理想值 2.0 以上；趋势模板仍成立；大盘没有大跌；没有重大利空。
6. 禁止追高：价格超过中枢点 5% 以上、突破 K 线走弱、大盘暴跌、未设止损时，不给买入结论。

## 止损与仓位

先定退出点，再谈买入。

- 初始止损：优先设在中枢点下方 1%-2%，或 VCP 最后一次收缩低点下方；形态不清晰时使用买入价下方 5%-8%。
- 单笔风险：通常控制在总资金的 1%-1.5%。
- 仓位公式：`仓位金额 = 单笔风险金额 / 止损幅度`。
- 禁止亏损加仓。只在已有浮盈时加仓，且加仓部分必须有独立止损。
- 分批买入：突破时先买计划仓位 50%；次日继续走强，或缩量回踩中枢点不破，再加 30%-50%。

## 持有与卖出流程

1. 止损卖出：触及止损线立即卖出。
2. 盈亏平衡：盈利达到初始风险的 2-3 倍时，把止损上移到成本附近。
3. 保护利润：涨幅达到 20%-25% 后，把止损上移到成本上方约 5%-8%。
4. 趋势跟踪：用 MA10 或近期波谷作为移动止损参考，避免从跟踪参考位回撤超过约 8%-10%。
5. 弱势卖出：跌破 MA10 且无法快速收回时减仓；跌破 MA20 再减；跌破 MA50、近期波谷或放量跌破中枢点/VCP 下沿时清仓。
6. 强势卖出：出现巨量长上影、连续大涨后放量滞涨、扩散形态、板块龙头见顶、估值严重透支或 50/80 法则警报时，主动止盈或分批卖出。
7. 50/80 法则：龙头股涨幅达到 50% 后收紧止损；涨幅达到 80% 后高度警惕，若出现见顶信号，至少考虑卖出一半。

## 输出格式

先给结论，再给依据。保持短、硬、可执行。

```text
结论：剔除 / 观察 / 买入触发 / 持有 / 减仓 / 卖出 / 数据不足
关键价位：中枢点、买入区间、止损位、加仓点、移动止损
依据：3-6 条，分别对应趋势、形态、成交量、市场环境、风险
失效条件：什么情况说明判断失效
下一步：一个具体动作，或需要补充的数据
```

数据不足时，只输出 `结论：数据不足`，并列出完成 SEPA 判断所需的最少缺失字段。
