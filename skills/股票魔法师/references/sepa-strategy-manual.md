## 一、策略概述

### 1.1 策略内核

SEPA（Specific Entry Point Analysis，特定入场点分析）的本质是：

> **在已经处于上升趋势（第二阶段）的强势股中，等待波动收窄、卖压枯竭的整理末端，于中枢点放量突破时精确买入，用严格止损保护资本，让利润在趋势延续中奔跑。**

一句话总结：**买强势股的强势突破点，截断亏损，让利润奔跑。**

### 1.2 盈利逻辑

| 要素 | 逻辑 |
|------|------|
| **为何只买第二阶段** | 上升趋势中做多，顺势而为，阻力最小 |
| **为何用VCP** | 波动收窄=卖压消化=筹码从弱手转向强手，突破后动能最大 |
| **为何中枢点买入** | 供需失衡的临界点，风险最低（止损距离小）、收益最大 |
| **为何严控止损** | 亏损以几何级对抗（亏20%需涨25%回本），小亏是成功的成本 |

### 1.3 业绩预期与参数定位（中短期）

| 参数 | 取值 | 说明 |
|------|------|------|
| 持仓周期 | 2周 ~ 3个月 | 趋势延续期间持有 |
| 平均成功率（目标） | 45% ~ 55% | Minervini 40-50%即可盈利 |
| 单笔收益目标 | 20% ~ 30% | 对应风险比 2:1 ~ 3:1 |
| 单笔最大风险 | 5% ~ 8% | 距买入价的止损幅度 |
| 单笔仓位风险 | 总资金的 1% ~ 1.5% | 单笔亏损不超过总资金1.5% |
| 持仓数量 | 3 ~ 6 只 | 集中于最强标的 |

### 1.4 适用的市场环境

- ✅ **趋势行情**（市场处于上升或结构性行情）——最佳
- ✅ **结构性行情**（部分板块/个股走强）——可做，聚焦领头羊
- ⚠️ **震荡市**——减少交易频次，提高筛选标准
- ❌ **系统性下跌/熊市**——空仓或极轻仓，等待新领头羊率先触底

## 二、选股筛选：趋势模板

### 2.1 趋势模板量化标准（第二阶段确认）

一只股票必须**同时满足以下全部条件**，才进入候选池。这是不可妥协的硬门槛。

| # | 条件 | 检查方式 |
|---|------|----------|
| 1 | 当前股价 **高于** 150日均线（MA150） | 日线 |
| 2 | 当前股价 **高于** 200日均线（MA200） | 日线 |
| 3 | MA150 **高于** MA200 | 日线 |
| 4 | MA200 呈 **上升趋势**（当前值 > 60天前的值） | 日线 |
| 5 | 当前股价 **高于** 50日均线（MA50） | 日线 |
| 6 | 股价 **不低于** 50日均线（不能大幅跌破） | 日线 |
| 7 | 股价距52周最低点 **至少上涨30%** | 确认非底部 |
| 8 | 股价在52周最高点的 **25%以内** | 确认接近新高、动量强 |
| 9 | 相对强度排名在前 **20%~30%** | 板块/全市场排名 |

> **关键**：第7、8条是Minervini的核心理念——**要成为赢家，必须创新高**。不抄底，只买已经走强、接近新高的股票。

### 2.2 基本面叠加筛选（提升胜率）

技术面是入场依据，基本面是持续性保障。在趋势模板通过的基础上，叠加：

| 维度 | 标准 |
|------|------|
| **盈利增长** | 最近一期净利润同比增速 ≥ 25%，且加速或维持 |
| **收入增长** | 营收同比增速 ≥ 20% |
| **EPS** | 正且增长 |
| **机构持仓** | 有基金/北向资金增持迹象 |
| **行业地位** | 细分龙头或行业领头羊 |
| **催化剂** | 有业绩超预期、新产品、政策利好等驱动 |

### 2.3 排除清单（直接剔除）

- ❌ ST、*ST 股票
- ❌ 日均成交额 < 3亿元（流动性不足）
- ❌ 长期停牌后复牌未满30个交易日
- ❌ 处于退市风险警示
- ❌ MA200 仍在下降（即使其他条件满足）
- ❌ 股价大幅低于MA50（弱势）

## 三、形态识别：VCP与买入形态

通过趋势模板筛选后，进入形态识别阶段。**VCP是首要、最核心的买入形态。**

### 3.1 VCP 波动收窄模式（核心形态）

#### 定义
股票在上升途中进入横盘整理，**价格波动幅度逐次收窄，成交量同步萎缩**，最终在中枢点放量突破。

#### 识别要点

| 要素 | 标准 |
|------|------|
| **所处阶段** | 必须处于第二阶段（趋势模板已确认） |
| **收缩次数** | 通常 2 ~ 5 次（越多越可靠） |
| **波动递减** | 每次回调幅度小于前一次（如 15%→10%→5%→2%） |
| **量能递减** | 每次收缩期间成交量逐步萎缩 |
| **最后一次收缩** | 波动极小（2%~5%），成交量极度萎缩 |
| **中枢点** | 最后一次收缩的上沿阻力位 |
| **突破确认** | 放量突破中枢点（量比 ≥ 1.5，最好 ≥ 2） |

#### VCP 时间框架
- 中短期持仓：VCP 整理时长通常 **3 ~ 8 周**
- 周线级别也应显示为横盘或缓涨

#### VCP 的"2T"简化判断
如果不会数收缩次数，用**两次回踩**简化判断：
1. 第一次回踩幅度较大（如 10-15%），量能放大
2. 第二次回踩幅度明显缩小（如 3-5%），量能萎缩
3. 第三次或直接放量突破前高 → 买点

### 3.2 其他辅助买入形态

当VCP不明显时，以下形态也可作为买点（但VCP优先）：

| 形态 | 特征 | 买点 |
|------|------|------|
| **杯柄形态** | 回调后形成U型底，右侧小幅回抽形成"柄" | 柄部上沿突破 |
| **双底** | 两次探底，第二底略高或持平 | 颈线突破 |
| **高位旗形** | 快速拉升后小幅横盘整理 | 旗形上沿突破 |
| **3-C（紧凑杯柄）** | 更紧凑的杯柄变体 | 突破点 |
| **欺骗（Shakeout）** | 突破前快速下杀洗盘后拉回 | 拉回后突破前高 |

### 3.3 危险形态（避免买入）

- ❌ **连续跳空缺口过多**：可能过度透支买盘
- ❌ **扩散形态**（波动越来越大）：供应增加，危险
- ❌ **巨量长上影**：抛压沉重
- ❌ **跌破VCP下沿**：形态破坏
- ❌ **低量上涨、高量下跌**：量价背离

## 四、买入执行：中枢点突破

### 4.1 中枢点（Pivot Point）定义

中枢点 = VCP 最后一次收缩的**上沿阻力位**，也是供需失衡的临界突破点。

### 4.2 买入触发条件（全部满足）

| # | 条件 | 说明 |
|---|------|------|
| 1 | 股价**突破中枢点** | 收盘价站上中枢点 |
| 2 | **成交量放大** | 突破当日量比 ≥ 1.5，理想 ≥ 2 |
| 3 | **趋势模板仍成立** | MA150 > MA200，股价在均线上方 |
| 4 | **市场环境配合** | 大盘非大跌日，板块有资金流入 |
| 5 | **无重大利空** | 排除业绩雷、突发利空 |

### 4.3 买入方式

**方式A：突破当日买入（激进，适合强势突破）**
- 突破中枢点且放量时，当日尾盘或次日开盘买入
- 适合：突破坚决、量能充沛、市场强势

**方式B：突破回踩买入（稳健，适合震荡市）**
- 突破后等待 1~3 天回踩中枢点（此时变为支撑）
- 回踩缩量、不破中枢点时买入
- 适合：突破力度一般、想降低假突破风险

**方式C：分批买入（推荐，平衡风险）**
- 突破当日买入计划仓位的 50%
- 确认有效（次日继续走强或回踩不破）后加仓 30%~50%
- 加仓前提：股价未跌破初始止损

### 4.4 买入禁区

- ❌ 大幅高开后追高（已脱离中枢点 5% 以上）
- ❌ 放量长阴线时买入
- ❌ 大盘暴跌日强行买入
- ❌ 未设止损就买入

## 五、风险管理与止损

### 5.1 核心原则

> **风险优先**：每一笔交易在买入前，必须先确定退出点（止损价）。

> **亏损严控**：单笔亏损不超过买入价的 5%~8%，且单笔风险不超过总资金的 1.5%。

### 5.2 止损设置

#### 初始止损（买入时设定）
| 方法 | 说明 | 适用 |
|------|------|------|
| **中枢点下方法（首选）** | 止损设在中枢点（VCP下沿）下方 1%~2% | 标准VCP买入 |
| **最近低点下方法** | 止损设在VCP最后一次收缩的低点下方 | 形态清晰时 |
| **固定比例法** | 买入价下方 5%~8% | 形态不明确时 |

> **计算示例**：买入价 20 元，中枢点 19.5 元，止损设 19.1 元（中枢点下方约 2%），单笔风险 = (20-19.1)/20 = 4.5%。

#### 交错止损（可选，分散择时风险）
若对买点把握不足，可分批止损：
- 亏损 3% 时减仓 1/3
- 亏损 5% 时减仓 1/3
- 亏损 8% 时清仓剩余

#### 移动止损（盈利后启动）

这是保护利润的核心机制，**分三阶段**：

| 阶段 | 触发条件 | 操作 |
|------|----------|------|
| **阶段一：保本** | 股价涨幅达 **2~3倍初始风险**（如风险5%，涨10~15%） | 止损上移至 **盈亏平衡点** |
| **阶段二：保利润** | 股价涨幅达 **20%~25%** | 止损上移至成本价上方 5%~8% |
| **阶段三：跟踪** | 股价持续上涨 | 止损跟随 **MA10 或最近波谷低点** 上移，不回撤超过 8%~10% |

### 5.3 止损纪律

- ⚠️ **止损线被触及，无条件执行**——不犹豫、不抱幻想、不"再等等"
- ⚠️ 止损后如果股价重新走强，**可以重新买回**（Minervini原话："如果事实证明是虚惊一场，我还可以再买回来"）
- ⚠️ 单日暴跌触及止损，**当日执行**，不等次日

### 5.4 风险预算表

| 总资金 | 单笔风险1% | 单笔风险1.5% | 单笔最大仓位（风险8%时） |
|--------|-----------|-------------|------------------------|
| 50万 | 5,000 | 7,500 | 6.25万 / 9.375万 |
| 100万 | 10,000 | 15,000 | 12.5万 / 18.75万 |
| 200万 | 20,000 | 30,000 | 25万 / 37.5万 |

> **仓位计算公式**：`单笔仓位 = 单笔风险金额 / 单笔止损幅度`
> 例：总资金100万，单笔风险1.5%（1.5万），止损幅度5%，则仓位 = 1.5万 / 5% = 30万。

## 六、仓位管理

### 6.1 集中持仓原则

- **持仓 3~6 只**，不宜过度分散
- Minervini 倾向集中：**把资金放在最强的 2~3 只股票上**
- 二换一原则：2 只表现较弱的股票换成 1 只最强的

### 6.2 加仓规则

**只在盈利时加仓，绝不摊平亏损。**

| 条件 | 操作 |
|------|------|
| 股票上涨，已有浮盈 | 可以加仓 |
| 加仓后总风险不超过原计划 | 确认 |
| 加仓部分有独立止损 | 必须设定 |
| 股票下跌/亏损 | **禁止加仓摊平** |

### 6.3 仓位调整（按市场强度）

| 市场状态 | 总仓位 | 说明 |
|----------|--------|------|
| 强势上行 | 80%~100% | 趋势明确，可满仓最强标的 |
| 结构性行情 | 50%~70% | 聚焦领涨板块 |
| 震荡整理 | 30%~50% | 减少交易，提高标准 |
| 系统性下跌 | 0%~20% | 空仓或极轻仓，等待新领头羊 |

## 七、卖出规则

### 7.1 卖出分类

| 类型 | 触发 | 执行 |
|------|------|------|
| **止损卖出** | 跌破止损线 | 立即、无条件 |
| **弱势卖出** | 跌破关键支撑/MA10 | 及时卖出 |
| **强势卖出** | 放量滞涨/见顶信号 | 主动获利了结 |
| **目标卖出** | 达到目标涨幅或50/80法则 | 分批止盈 |

### 7.2 强势卖出信号（主动止盈）

出现以下信号时，主动卖出锁定利润：

| 信号 | 说明 |
|------|------|
| **放量巨量长上影** | 顶部抛压，主力出货迹象 |
| **连续大涨后放量滞涨** | 动能衰竭 |
| **跌破MA10（短线）** | 短期趋势转弱 |
| **市盈率过度扩张** | 估值透支（结合基本面） |
| **出现扩散形态** | 波动加大，供应增加 |
| **50/80法则触发** | 涨幅达50%或80%，高度警惕 |
| **板块龙头见顶** | 板块整体走弱 |

### 7.3 弱势卖出信号（防守卖出）

| 信号 | 操作 |
|------|------|
| 跌破MA10且无法快速收回 | 减仓 1/3~1/2 |
| 跌破MA20 | 再减仓 |
| 跌破MA50 或 跌破最近波谷 | 清仓 |
| 放量跌破VCP下沿/中枢点 | 立即清仓 |

### 7.4 50/80 法则

> 一旦长期领头羊股价到达顶端，**50%概率下跌80%，80%概率下跌50%**。

**操作含义**：
- 涨幅达 **50%**：提高警惕，收紧止损至 MA10 或最近低点
- 涨幅达 **80%**：高度警戒，考虑分批止盈（至少卖出 1/2）
- 出现任何见顶信号：果断卖出

### 7.5 盈亏平衡法则

- 股价涨至 **3倍初始风险** 时（如风险5%，涨15%），止损上移至**盈亏平衡点**
- 此后这笔交易**不可能亏损**，心态从容

### 7.6 卖出心态

> "你总会卖得太早或太晚——接受这个事实。专注于执行计划，而不是追求完美卖点。"

- 不要因为卖出后股票继续涨而后悔
- 不要因为没卖在最高点而自责
- **执行计划 > 追求完美**

## 八、交易记录与复盘

### 8.1 为什么记录

> "让自己保持诚实——记录你的每一笔交易，包括失误和机会成本。"

Minervini 强调：**专业交易者的标志是系统记录每笔交易**，用数据验证策略，而非凭直觉。

### 8.2 记录要素（投资三角）

每笔交易记录三维度：

| 维度 | 内容 |
|------|------|
| **预期** | 买入逻辑、形态、中枢点、止损、目标 |
| **执行** | 实际买入价、量、时间、是否按计划 |
| **结果** | 卖出价、盈亏、持有天数、复盘反思 |

### 8.3 月度复盘指标

| 指标 | 说明 |
|------|------|
| 交易次数 | 频率是否合理 |
| 胜率 | 盈利交易 / 总交易 |
| 平均盈利 / 平均亏损 | 盈亏比 |
| 最大单笔盈利 / 最大单笔亏损 | 极值 |
| 总收益率 | 月度表现 |
| 最大的失误 | 复盘改进点 |
| 最好的交易 | 总结可复制的经验 |

### 8.4 复盘问题清单

每笔交易结束后问自己：
1. 是否严格按计划执行？
2. 买入时趋势模板是否全部满足？
3. VCP形态是否标准？
4. 止损是否及时？
5. 卖出是否按规则？
6. 有没有情绪化操作？
7. 下次如何改进？

## 九、完整决策流程

```
【第一步：选股筛选】
  趋势模板9条件 → 全部满足？
    ├ 否 → 剔除
    └ 是 → 叠加基本面筛选 → 通过？
              ├ 否 → 剔除
              └ 是 → 进入候选池

【第二步：形态识别】
  候选股 → 日线是否出现VCP？
    ├ 否（其他形态）→ 评估辅助形态
    └ 是 → VCP是否标准（收缩递减+量能萎缩）？
              ├ 否 → 等待或剔除
              └ 是 → 确定中枢点

【第三步：买入执行】
  等待 → 放量突破中枢点？
    ├ 否 → 继续等待
    └ 是 → 检查：市场环境+无利空+趋势模板仍成立？
              ├ 否 → 放弃
              └ 是 → 计算仓位（风险1%~1.5%）→ 设定止损 → 买入

【第四步：持有与风控】
  持有 → 每日检查：
    ├ 跌破止损 → 立即止损
    ├ 涨至2~3倍风险 → 止损上移至盈亏平衡
    ├ 涨至20%~25% → 止损上移保护利润
    └ 趋势完好 → 继续持有，跟踪止损上移

【第五步：卖出】
  ├ 止损卖出（破位）→ 执行
  ├ 弱势卖出（跌破MA10/支撑）→ 减仓
  ├ 强势卖出（滞涨/见顶）→ 获利了结
  └ 50/80法则触发 → 分批止盈

【第六步：复盘记录】
  记录预期/执行/结果 → 月度统计 → 优化策略
```

## 附录：A股市场适配注意事项

1. **涨跌停板**：触及涨停时无法买入，可挂单排队或等待次日；跌停时无法卖出，需提前设止损避免被动
2. **T+1 交易**：当日买入次日才能卖出，买入时机更需谨慎，避免尾盘追高
3. **板块轮动**：A股板块效应强，优先选择当前主流资金关注的领涨板块龙头
4. **北向资金**：可参考北向资金流向作为机构资金动向的辅助指标
5. **财报季风险**：财报披露前后波动加大，Minervini建议如无十足把握，财报前可减仓规避
6. **新股次新股**：上市未满60个交易日的次新股，MA200等指标不稳定，需谨慎
