# 行业分析 (/行业分析)

触发 `行业分析` skill，对 `$ARGUMENTS` 指定的行业问题做分析。

## 必须做的事

1. **立即读取并遵循** `行业分析` skill 的 `SKILL.md`：
   - 项目级：`skills/行业分析/SKILL.md`
   - 全局：`~/.claude/skills/行业分析/SKILL.md`、`~/.codex/skills/行业分析/SKILL.md`、`~/.cursor/skills-cursor/行业分析/SKILL.md` 或对应 Agent 的 skills 目录
2. 先读取 `价值投资` skill 里的 `references/mcp-tool-guide.md`，按其中规则锚定数据时效：
   - 项目级：`skills/价值投资/references/mcp-tool-guide.md`
   - 全局：`~/.claude/skills/价值投资/references/mcp-tool-guide.md`、`~/.codex/skills/价值投资/references/mcp-tool-guide.md`、`~/.cursor/skills-cursor/价值投资/references/mcp-tool-guide.md`
3. 优先使用同花顺问财相关 skills：
   - `hithink-industry-query`
   - `hithink-sector-selector`
   - `hithink-market-query`
4. 若已安装社区 skills，可增强使用：
   - `行业轮动监控`
   - `产业链解读`
   - `捕捉公司事件机会`
5. 先给结论，再给依据；数据不足时直接说明缺什么。

## 使用方法

```text
/行业分析 未来6到12个月，A股哪些行业景气度改善、资金持续流入、估值仍处中低位？
/行业分析 按照宏观周期和利率环境，当前最可能跑赢的3个行业是什么？
/行业分析 请解读光模块产业链，找出利润弹性最大和卡脖子环节。
/行业分析 近30天创新药有哪些政策、订单、并购、扩产催化？
/行业分析 在储能行业中，筛选营收增速改善、ROE较高、机构预期上修、股价未大涨的A股公司。
```

## 不要做的事

- 不要跳过时间口径检查。
- 不要把题材热度直接当行业机会。
- 不要在字段缺失时补猜。
