# 股票比较 (/股票比较)

触发 `股票比较` skill，对 `$ARGUMENTS` 指定的多只股票做综合比较。

## 必须做的事

1. **立即读取并遵循** `股票比较` skill 的 `SKILL.md`：
   - 项目级：`skills/股票比较/SKILL.md`
   - 全局：`~/.claude/skills/股票比较/SKILL.md`、`~/.codex/skills/股票比较/SKILL.md`、`~/.cursor/skills-cursor/股票比较/SKILL.md` 或对应 Agent 的 skills 目录
2. 同时读取并遵循 `价值投资` 与 `股票魔法师` skill。
3. 对每只股票拉取或要求补充最新行情、财报/估值、均线、成交量、52 周高低点、相对强度、板块/大盘环境。
4. 输出机会成本排序、逐只股票依据、关键价位和买入/持有/减仓/卖出/换仓计划。

## 使用方法

```text
/股票比较 600519.SH 09992.HK AAPL
/股票比较 持有 600519.SH 成本 1500 仓位30%；候选 09992.HK AAPL；现金20%
```

## 不要做的事

- 不构成投资建议，不承诺收益。
- 数据不足时不要猜买点、止损或换仓比例。
- 不为短期热点强行换仓；只有风险收益明显更优时才给换仓结论。
