# 股票魔法师 (/股票魔法师)

触发 `股票魔法师` skill，对 `$ARGUMENTS` 指定的股票执行 SEPA / VCP 交易分析。

## 必须做的事

1. **立即读取并遵循** `股票魔法师` skill 的 `SKILL.md`：
   - 项目级：`skills/股票魔法师/SKILL.md`
   - 全局：`~/.claude/skills/股票魔法师/SKILL.md`、`~/.codex/skills/股票魔法师/SKILL.md`、`~/.cursor/skills-cursor/股票魔法师/SKILL.md` 或对应 Agent 的 skills 目录
2. 围绕 `$ARGUMENTS` 拉取或要求补充当前行情、均线、成交量、52 周高低点、相对强度、板块环境和基本面数据。
3. 按 skill 流程判断：趋势模板 → VCP/买入形态 → 中枢点突破 → 止损与仓位 → 持有/卖出信号。
4. 输出：结论、关键价位、依据、失效条件、下一步。

## 使用方法

```text
/股票魔法师 贵州茅台
/股票魔法师 600519.SH
/股票魔法师 AAPL
```

## 不要做的事

- 不构成投资建议，不承诺收益。
- 数据不足时不要猜测买卖点，先输出缺失字段。
- 不分析债券、基金、ETF、期货、期权或加密货币等非股票标的。
